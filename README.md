# Sentiment Analysis Web App (Django)

A web application built using Django and TextBlob that performs sentiment analysis on user-entered text.

## Features
- Classifies text as Positive, Negative, or Neutral
- Shows polarity as a visual bar
- Shows subjectivity label
- Dark mode UI
- Collapsible score card for raw scores

## Tech Stack
- Python
- Django
- TextBlob
- HTML, CSS, Bootstrap,Javascript

## How to Run

Using Command prompt or Terminal
python -m venv venv
source venv/bin/activate  # macOS/Linux
venv\Scripts\activate     # Windows

pip install django textblob
python manage.py runserver
